# Spotistats

Privacy-first Spotify streaming history analyzer.
Upload your Extended Streaming History JSON, get detailed stats.
Data never leaves your browser.

## Features

- Total play time, top artists/albums/tracks
- Time range filtering
- Peak activity detection
- Works offline, no account needed

## Get your data

1. Go to [Spotify Account privacy settings](https://www.spotify.com/account/privacy/)
2. Request "Extended streaming history"
3. Wait for email (few days)
4. Upload JSON files

## Tech

SolidJS, Effector, IndexedDB, Vite
